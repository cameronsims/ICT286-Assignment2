How to Search

Click on the "catalog" menu option, there you will be given two ways to search the Database for items. Searching by category or searching by keywords, both can be used seperately or together. Searching with "None" as a category will return all items. If no items have been queried the catalog entries will be empty, and no items will be visible until returned.

Cart Issues

To add to your cart, simply click on the product you'd like to order and click "Add to Cart." After this is done, you should be able to go to your cart. If you cannot order please try the following:
  • Creating / Logging into an Account.
  • Checking if the items you want are avaliable.
  • Contacting support staff if all this fails

Account Management

How to Create an Account

To create an account simply click on the "Log-in" button at the top right. If the button already displays a name, you have already been logged in as a verified user. If you are for whatever reason redirected from the log-in sign-up page, please try again.

How to Delete an Account

To delete an account, go to the log-in screen and it should be a red button that says "delete account."
Once an account is deleted it can not be recovered!

How to Change details on an Account

To change your account details please click on the name at the top right, it should send you to the user page where you are free to change your details.
